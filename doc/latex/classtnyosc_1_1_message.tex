\subsection{Detailed Description}
This class represents an Open Sound Control message. It supports Open Sound Control 1.0 and 1.1 specifications and extra non-\/standard arguments listed in \href{http://opensoundcontrol.org/spec-1_0.}{\tt http://opensoundcontrol.org/spec-\/1\_\-0.}
