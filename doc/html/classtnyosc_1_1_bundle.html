Detailed Description

This class represents an Open Sound Control bundle message. A bundle can contain any number of Message and Bundle.

Constructor & Destructor Documentation

tnyosc::Bundle::Bundle ( ) [inline]

Creates a OSC bundle with timestamp set to immediate. Call set_timetag to set a custom timestamp.
